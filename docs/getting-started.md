# Getting Started

[← Back to Documentation Index](index.md)

## Quick Start

1. **Install Python 3.8+**
2. **Navigate to project**: `cd QuizPython`
3. **Create virtual environment**:
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # Windows: venv\Scripts\activate
   ```
4. **Install dependencies**: `pip install -r requirements.txt`
5. **Run**: `python main.py`

### First Run

1. **Test Select Dialog** appears
2. **Choose quiz** from "New Test" tab
3. **Configure Options** (optional):
   - **Shuffle questions**: Randomize question order
   - **Practice mode**: No timer, free navigation between questions
   - **Hide answers during quiz**: Answers hidden until quiz completion
4. **Start Test**
5. **Begin answering** questions

### Basic Usage

- **Select answer**: Click option or press 1-5
- **Confirm**: Click "Confirm Answer" or press Space
- **Navigate**: ←/→ arrows or Previous/Next buttons
- **Review**: After completion, review incorrect answers
- **Pause**: Ctrl+P (hides questions, pauses timer)
- **Dark Mode**: Ctrl+D to toggle theme

### Quiz Options

When starting a new test, you can enable these options:

- **Shuffle Questions**: Randomizes question order for varied practice
- **Practice Mode**: Disables timer and allows free navigation (no answer confirmation required)
- **Hide Answers During Quiz**: Keeps answers hidden until quiz completion (useful for exam simulation)

See the main README for detailed information about each option.
