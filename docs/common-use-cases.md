## Use Case 1: Study for Exam

1. Start new quiz from "New Test" tab
2. Answer questions at your pace
3. Review mistakes after completion
4. Click "Study These Questions" to retake wrong ones

### Use Case 2: Resume Session

1. Open "Previous Sessions" tab
2. Select session to resume
3. Click "Resume Session"
4. Continue where you left off

### Use Case 3: Review Past Mistakes

1. Click "Review Incorrect Answers" tab
2. Select completed quiz
3. Review mode entered automatically
4. Study incorrect answers with highlighting
5. Click "Study These Questions" to practice again

### Use Case 4: Shuffled Practice

1. Enable "Shuffle questions" checkbox
2. Start quiz - questions in random order
3. Get different order each time

### Use Case 5: Hide Answers During Quiz

1. Enable "Hide answers during quiz" checkbox
2. Start quiz and answer all questions
3. Answers remain hidden during the quiz (no immediate feedback)
4. At completion, all answers are automatically revealed
5. Navigate through questions to review your selections alongside correct answers
6. Useful for simulating real exam conditions where you don't get immediate feedback

### Use Case 6: Practice Mode Study

1. Enable "Practice mode (no timer, free navigation)" checkbox
2. Start quiz - timer is disabled
3. Navigate freely between questions without answer confirmation
4. Use "Show Answer" button for immediate feedback on each question
5. Jump to specific questions using the question timeline or number keys (1-9)
6. Study at your own pace without time pressure
7. Great for initial learning and exploration

### Use Case 7: Combined Options

**Exam Simulation**:
1. Enable both "Shuffle questions" and "Hide answers during quiz"
2. Get randomized question order with no feedback until completion
3. Most realistic exam simulation

**Flexible Learning**:
1. Enable "Practice mode" and "Shuffle questions"
2. Study material in different orders without time pressure
3. Jump between questions freely to compare concepts

### Use Case 8: Track Progress

1. Complete multiple quiz sessions
2. Check `results/` directory for JSON files
3. Analyze accuracy and completion rates
4. Focus on frequently incorrect question types
